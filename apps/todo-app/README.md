# 📝 تطبيق قائمة المهام | Todo App

تطبيق ويب أنيق وسهل الاستخدام لإدارة مهامك اليومية مع حفظ تلقائي باستخدام LocalStorage.

## ✨ المميزات الرئيسية

- ✅ **إضافة المهام** - أضف مهام جديدة مع الوصف والأولوية والتاريخ
- 🎯 **تحديد الأولوية** - صنف المهام حسب الأولوية (عالية، متوسطة، منخفضة)
- 📅 **تواريخ الاستحقاق** - حدد موعد نهائي لكل مهمة
- 🏷️ **الفئات** - صنف مهامك حسب الفئات
- 💾 **حفظ تلقائي** - جميع المهام محفوظة في LocalStorage
- 🎨 **واجهة جميلة** - تصميم عصري مع تأثيرات Glass Morphism
- 📊 **إحصائيات** - تتبع عدد المهام والمكتملة والنشطة
- 🔍 **تصفية ذكية** - صفي المهام حسب الحالة والأولوية
- 📱 **متجاوب** - يعمل على جميع الأجهزة والشاشات

## 🛠️ التقنيات

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React
- LocalStorage API

## 🚀 البدء السريع

```bash
cd apps/todo-app
npm install
npm run dev
```

## 📖 الاستخدام

1. **إضافة مهمة**: اضغط على "إضافة مهمة جديدة"
2. **تحديد كمكتملة**: اضغط على دائرة المهمة
3. **حذف**: اضغط على زر الحذف
4. **تصفية**: اختر من خيارات التصفية

## 💾 الحفظ التلقائي

جميع المهام محفوظة في LocalStorage بجهازك - لا تحتاج إنترنت!
